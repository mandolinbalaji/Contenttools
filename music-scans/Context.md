The "Golden Rules" of BrahmaLayamAtomic Notes: Every character typed is an object { char: 'S', octave: 0, speed: 1 }. Never store just raw text; it makes the ^ line and Western Staff sync impossible.Laya Calculation: * $Total Pulses = \sum (1 / Speed)$Vertical lines are drawn based on the cumulative pulse count matching the Jathi value.Sruthi Mapping: Map 'S' to the frequency defined by the user's Sruthi choice (e.g., C4 = 261.63 Hz). Use a standard Carnatic frequency ratio table for other notes.Backend (app.py):GET /lessons: Returns a list of filenames from a data/ folder.POST /save: Writes the JSON object to a file.POST /export: Converts JSON to MusicXML format
